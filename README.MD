# Cambio com Python
Um script para pegar os valores, via requeisição web, das cotações das principais moedas mundiais:
  - Dollar
  - Euro
  - Bitcoin

API utilizada: [FIXER.IO](https://fixer.io/)

### Biblioteca para funcionamento do script
```sh
pip install requests
pip install json
pip install pandas
```
